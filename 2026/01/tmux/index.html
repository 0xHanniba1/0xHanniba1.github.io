tmux命令

Posted at 2026-01-05

Tmux 常用命令速查表

核心机制：前缀键 (Prefix Key)

所有快捷键操作前，必须先按下“扳机”：Ctrl + b (松开后再按后续按键)。

1. 会话管理 (Session Management)

在普通终端 (Terminal) 中执行：

动作 命令 说明
新建会话 tmux new -s <名字> 创建一个有名字的房间 (例如 tmux new -s claude)
列出会话 tmux ls 查看后台有哪些房间在运行
回到会话 tmux attach -t <名字> 重新连接回指定的房间 (简写 tmux a -t <名字>)
彻底删除 tmux kill-session -t <名字> 在外部强制关闭某个房间

在 Tmux 内部操作：

动作 快捷键/命令 说明
暂时离开 前缀键 + d Detach。人走，房间留着，程序继续跑
彻底结束 输入 exit 或 Ctrl+d 关门熄灯，结束当前面板/会话

2. 屏幕切分 (Pane Management)

注意：操作前需先按前缀键 Ctrl + b

动作 快捷键 记忆法
左右切分 % (Shift + 5) 斜杠 / 把屏幕分两半
上下切分 " (Shift + ') 引号上下两点，代表上下分
切换光标 方向键 (⬆️⬇️⬅️➡️) 想去哪边按哪边

3.设置鼠标移动!!!

tmux set -g mouse on 回车后，你就可以直接用鼠标滚轮滚动查看历史了，点击面板也会自动切换焦点
